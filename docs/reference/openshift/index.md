# Bienvenue sur la Carte de la Documentation OpenShift

Ce site a pour but de vous fournir une **carte de navigation** claire et structurée pour la documentation officielle de **Red Hat OpenShift Container Platform (OCP)**.

Que vous soyez administrateur, architecte, développeur ou simplement curieux, ce guide vous aidera à :

- **Comprendre** la structure de la documentation officielle.
- **Maîtriser** les concepts et compétences clés du produit.
- **Optimiser** vos coûts grâce à une meilleure compréhension du licensing.

Naviguez à travers les différentes sections pour explorer le contenu.
