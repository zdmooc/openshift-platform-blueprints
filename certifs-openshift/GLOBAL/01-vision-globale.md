# Vision globale

Le sens de ce dépôt n'est pas de collectionner des badges.

L'idée est de construire une trajectoire cohérente :
- fondations ;
- administration ;
- spécialisation ;
- architecture ;
- crédibilité marché.

Les certifications doivent servir :
- un positionnement professionnel ;
- une progression technique ;
- une capacité à démontrer des choix d'architecture.
