# 01A — Projects and contexts

## Objectif
Créer et gérer plusieurs projets, vérifier le projet courant et lire les ressources principales.

## Commandes
```bash

oc new-project ex288-01a || oc project ex288-01a
oc new-project ex288-01a-b || true

oc whoami
oc projects
oc project
oc get all

```

## Vérifications
- deux projets visibles
- bon projet courant identifié
